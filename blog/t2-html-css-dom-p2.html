Technical Blog Part 2


What is grid based design?


Grid based design is laying out your structure through the process of applying grids to position your elements. Rows and columns can be used to structure content accordingly with even spacing.


Whats all the hype about responsive webpages?


As technology is progressing, more and more people are using devices other than desktop computers to access the internet. This means devices like laptops, tablets and smartphones are starting to be used more and more, and they need a different layout of the webpage in order for the user to be able to use the site effectively. Responsive design allows developers to do this by making content responsive to what device the user is using, and the corresponding resolution given.


What is semantic structure?


Semantic structure is giveing element tags a particular name depending on what content it contains, or what purpose it is for. These include tags such as form, img or table to name a few.


Any other points?


The grid based design is a really useful way to effectively make a website responsive, depending on what content is has in it.
